# Weather Intelligence Dashboard

## Project Overview

This project is a Python-based weather dashboard that retrieves live weather information using a weather API. Users can search for a city, view the current temperature and weather condition, and save their search history.

## Features

- Search weather by city name
- Display current temperature
- Display current weather condition
- Save search history
- View previous searches
- Error handling for invalid input and missing files

## Technologies Used

- Python
- Requests
- APIs
- JSON
- File Handling
- Exception Handling

## Learning Outcomes

This project helped me learn:

- How APIs work
- How to make HTTP requests using Python
- How to process JSON data
- How to use file handling to store information
- How to implement error handling
- How to build a menu-driven application
